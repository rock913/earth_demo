二、 核心技术架构升级与难点攻克

补充：服务器访问与端口约定（47.245.113.151）

- 前端访问入口：`http://47.245.113.151:8502/`
  - 前端为 Vite Dev Server，需监听 `0.0.0.0:8502` 才能被外网访问（已在 `vite.config.js` 与启动脚本中设置）。
  - 安全组/防火墙需放行 8502 端口。
- 后端默认：`127.0.0.1:8503`
  - 推荐通过前端的 `/api` 反向代理访问后端：浏览器请求 `http://47.245.113.151:8502/api/...`，由服务器侧转发到 `http://127.0.0.1:8503`，避免“浏览器访问自己本机 127.0.0.1”的常见坑。
  - 如需直接暴露后端到外网：设置 `API_HOST=0.0.0.0`，并放行 8503；同时注意 CORS 与安全策略。

补充：AI 图层颜色含义（直观解读优先）

- 地表 DNA（RGB 语义纹理）
  - 颜色本身不代表“风险大小”，而是语义通道（A00/A01/A02）组合后的地物纹理差异。
  - 更重要的是“边界与突变”：颜色突变、纹理破碎处往往对应功能区边界、混合地表或强人类活动。
- 变化雷达（红→橙→白）
  - 表示跨期语义距离的强度梯度：越亮（趋近白）变化越强，优先级越高。
  - 本质是“变化强度热区图”，用于快速圈定结构性变化热点。
- 建设强度（黑→蓝→青→白）
  - 表示建设强度从低到高：越亮（青/白）通常对应更高密度、更连续的人造地表。
  - 适合用于识别连片开发、扩张边界与重点建设廊道。
- 生态韧性（黑→深绿→亮绿→黄绿）
  - 表示胁迫/退化信号从弱到强：越偏黄绿，越需要优先复核与跨期对比验证。
  - 适合用于做“风险巡检清单”，而不是直接做最终定性结论。

痛点 1：地图卡顿与黑块 (渲染与并发瓶颈)

重构前： 使用 Streamlit/Folium，服务端渲染导致视角一动就卡死；后端 urllib 同步阻塞导致高并发请求 Timeout（出现黑块）。

重构后： * 前端： 引入 CesiumJS (WebGL)，利用客户端 GPU 硬件加速，实现 60 帧丝滑的 3D 飞行与缩放。

后端： 使用 FastAPI + httpx (异步非阻塞) + LRU 内存缓存。瞬间吞吐上百个瓦片请求，并将请求过的瓦片缓存在服务器内存中，彻底消灭黑块与白屏。

痛点 2：AI 图层视野受限 (Clip 裁剪问题)

重构前： 引擎强制裁剪中心点 20km 范围，导致缩小地图时 AI 图层只有“一小块方块”。

重构后： 解除空间禁锢。移除后端的 .clip(region) 算子，利用 GEE 的 .mosaic() 在云端动态拼接覆盖当前视锥体（Frustum）的所有数据，实现全域无缝铺满。

痛点 3：缺乏“业务闭环” (图表与决策缺失)

重构前： 只有干瘪的阈值滑块和热力图。

重构后： 结合 LLM 大语言模型，实现结构化输出：
[异动感知 Observation] -> [归因分析 Reasoning] -> [行动建议 Action]
将高维数学特征直接翻译为符合政府公文语境的决策简报。

三、 UI 深度美化与开发技巧 (UI Polish Tips)

1. 隐藏 Cesium 默认 Logo 及版权信息

为了保持指挥舱界面的极致纯净（去除商业化水印），在 Vue3 的 <style scoped> 中使用深度选择器隐藏底部栏：

/* 核心修复：隐藏 Cesium Logo 及底部版权信息，打造沉浸式纯净 UI */
:deep(.cesium-viewer-bottom) {
  display: none !important;
}


(注：在正式商用环境中，请确保符合 Cesium 的开源/商用授权协议协议；内部 PoC 演示可直接隐藏)

2. 防止 3D 地球变为“无贴图网格”

使用免费且无需 Token 的高清卫星底图直连方案，替代容易 401 报错的默认 Ion 资产：

imageryProvider: new Cesium.UrlTemplateImageryProvider({
    url: '[https://server.arcgisonline.com/ArcGIS/rest/services/World_Imagery/MapServer/tile/](https://server.arcgisonline.com/ArcGIS/rest/services/World_Imagery/MapServer/tile/){z}/{y}/{x}',
    maximumLevel: 18
})


3. 关闭黑夜模式 (防止中国区全黑)

Cesium 会模拟真实系统时间的光照。如果在晚上开发演示，需强制关闭真实光照：

viewer.scene.globe.enableLighting = false; // 关掉夜晚黑屏
viewer.scene.fog.enabled = true;           // 开启大气层散射科幻光晕
